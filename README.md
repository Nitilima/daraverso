# Boas vindas!

Oi! Eu sou o Arin, e desenvolvi esse site juntamente com minha esposa. Se você está aqui, provavelmente está com curiosidade de como o site funciona ou quer tirar inspiração dele. O design foi feito inteiramente por mim, com ajuda do Coolors para definir uma paleta de cores. Já o desenvolvimento, deixei para minha esposa e algumas IAs para agilizar o processo. 

O site foi desenvolvido com banco de dados SQL, linguagens back-end PHP e front-end React. A biblioteca de componentes utilizada foi a [Shadcn](https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://ui.shadcn.com/&ved=2ahUKEwjCx4CJ7YmUAxVHpJUCHX6kNcwQFnoECA4QAQ&usg=AOvVaw26SVeBbB0mrlJEGtTeLGQW).

**Não, em nenhum processo criativo foi utilizado IA.**
